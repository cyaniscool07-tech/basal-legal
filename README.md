# basal-legal

Public hosting for the **Basal** app's legal pages (privacy policy), so
Apple/Google can link to a public URL without the main app repository
(private, source code) needing to be public.

Source of truth for the policy text is `PRIVACY_POLICY.md` in the main
`Basal-App` repository — this repo only hosts the rendered static page
via GitHub Pages. Keep `index.html` in sync manually when the policy
changes there.
